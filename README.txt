1. Biblioteki:

	libgtk-3 (wersja 3.24.20)
	libnotify (wersja 0.7.9)


2. Podstawowe funkcjonalności:
	
	GUI:
	    - toolbar z tooltipami
	    - tabela wyświetlająca wprowadzone wydarzenia
	    - sortowanie po dacie, czasie lub nazwie wydarzenia
	    - wyskakujące okno informujące o nadchodzącym wydarzeniu
	    - wyskakujące okno informujące o przeszłym wydarzeniu, 
	      które nie zostało odczytane w czasie, który został zapisany
	
	DODAJ:
	    - wprowadzenie nowego wydarzenia
	    - blokada wprowadzenia roku wcześniejszego niż bieżący
	    - sprawdzenie poprawności wprowadzanej daty (rok przestępny etc.)
	    - sprawdzenie czy podana data i czas nie jest z przeszłości
	    - popup w przypadku wprowadzenia niepoprawnej daty lub czasu
	
	EDYTUJ:
	    - edycja wybranego wydarzenia
	    - możliwość zachowania oryginalnej wiadomości
	    - sprawdzanie daty i czasu jak w przypadku funkcji wyżej
	    
	USUŃ:
	    - usunięcie wybranego wydarzenia
	    
	INFO
	
	WYJŚCIE
	

3. Testowano na Ubuntu 20.04

4. W katalogu dodałam też skrypt do kompilacji i instalacji bibliotek
